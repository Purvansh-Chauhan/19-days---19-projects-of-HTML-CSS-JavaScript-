# Quiz Game

An interactive quiz game built with HTML, CSS, and JavaScript.

## Features

- Multiple choice questions
- Real-time score tracking
- Progress bar indicator
- Visual feedback for correct/incorrect answers
- Result screen with performance message
- Restart functionality

## How to Play

1. Click "Start Quiz" button
2. Select answers for each question
3. Get instant feedback on your answers
4. View your final score and performance message
5. Click "Restart Quiz" to play again

## Technologies Used

- HTML5
- CSS3
- JavaScript (ES6)
